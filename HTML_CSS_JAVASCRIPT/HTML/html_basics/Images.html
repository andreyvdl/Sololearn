The "img" tag doesn't have a closing tag and it has an attribute "src="image_location" ";
You can also add a "alt="description" " to describe a image that don't load;
You can use "height="size_in_px_or_%" and width="size_in_px_or_%" " to define the image size, but big images takes more time to load;
And finally, you can add a "border="size_in_px_or_%" " to make a border outside the image.
