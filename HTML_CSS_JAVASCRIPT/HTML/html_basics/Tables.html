The tags "table, /table" determine the start and end of a table;
You can add a "border" attribute to the table;
The tags "tr, /tr" determine the start and the end of a table row;
The tags "td, /td" determine the content of a table column, inside they you can have texts, images, lists, other tables, etc;
You can add a "colspan" attribute to merge two columns on one cell, with "rowspan" you can do the same with the row;
Attributes like "bgcolor and align" works too.
