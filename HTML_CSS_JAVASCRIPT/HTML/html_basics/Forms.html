The tag "form, /form" create a form container;
The attribute "action" will send the user to the url after it send the form;
The "method" attribute have two values, "GET and POST";
GET will make the form data visible on the page address;
POST is more secure for things like passwords, because it will upload the form;

The "input" tag will receive a input;
It "type" attribute can be a text, a password, radio, URL, submit, etc;
The "name" attribute specifies the name for a form.

After the form is submitted it will be sent to the server with the use of a programming language like PHP.
