Some Important Message

HTML elements are defined as block elements and inline elements;
Block elements start from a new line, they are:
  • h1
  • form
  • li
  • ol
  • ul
  • p
  • pre
  • table
  • div

Inline elements works without a line break, they are:
  • b
  • a
  • strong
  • img
  • input
  • em
  • span

The "div, /div" tag is a block element, and is used as a container for other elements;
Similar to div, "span, /span" is a tag used as a container, but this one is a inline element;
You can have a div element inside of a div element, as a span element inside of a span element, BUT never a block element inside a inline element.
The attribute "style" change things related to the element it is on, like the e.g. changes the color of the font, the background color and the padding of the element inside.

Other elements can be used as block elements or inline elements like:
  • APPLET - embed Java applet
  • IFRAME - inline frame
  • INS - inserted text
  • MAP - image map
  • OBJECT - embedded object
  • SCRIPT - script within an HTML document
