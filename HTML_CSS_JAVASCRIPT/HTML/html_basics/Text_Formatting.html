Bold text makes it bold, it's defined by "b, /b".
Big text makes it big, it's defined by "big, /big".
Italic text makes it italic, it's defined by "i, /i".
Small text makes it small, it's defined by "small, /small".
Strong text makes it bold and important, it's defined by "strong, /strong".
Subcripted text makes it subscripted, it's defined by "sub, /sub".
Superscripted text makes it superscripted, it's defined by "sup, /sup".
Inserted text makes it underlined, it's defined by "ins, /ins".
Deleted text makes it strikethrough, it's defined by "del, /del".

There's also emphasized text, makes it italic and important, it's defined by "em, /em".
