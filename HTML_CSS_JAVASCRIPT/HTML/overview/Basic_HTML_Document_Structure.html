Every HTML code start with the opening and closing tag "html, /html";
They are like the buns of a hamburger, you can't make one with it.

After that we have the opening and closing tag "head, /head";
On the head tag we put non-visual elements.

The next is the opening and closing tag "body, /body";
This is where the page in self is, headings, paragraphs, lists, links, buttons, images, inputs and etc.
