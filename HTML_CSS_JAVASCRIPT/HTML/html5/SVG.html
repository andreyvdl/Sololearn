SVG means Scalable Vector Graphics;
It's a image that is not based on pixel, so you can scale it without losing quality;
The tag "svg, /svg" defines the area to create shapes with svg;
The "cicle, /circle" tag, creates a circle, the attribute "cx" defines the Horizontal location, the "cy" attribute defines the Vertical location, the "r" attribute defines the radius, the attribute "fill" defines the color of the shape and "stroke" attribute defines the outline color;

Every element and attribute of a SVG can be animated;
The "rect, /rect" tag creates a rectangle, the attributes "x and y" defines the position;
The "line, /line" tag creates a line, the attributes "x1 and y1" defines the initial position, the attributes "x2 and y2" the final position, "stroke" defines the color, "stroke-linecap" defines the heads of the line and "stroke-width" defines how thick the line is;

The "polyline, /polyline" tag creates a polygonal line, "fill" attribute defines if it will be a color between the points, "points" defines the points of the line the first value is X the second is Y you use a comma to separate the points;
The "ellipse, /ellipse" tag creates a ellipse, the "rx" attribute defines the horizontal radius, the "ry" attribute defines the vertical radius;
The "polygon, /polygon" creates a polygon;
