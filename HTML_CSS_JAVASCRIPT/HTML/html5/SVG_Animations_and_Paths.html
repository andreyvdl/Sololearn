The tag "animate, /animate" animates the svg form that they are inside, their attributes are:
attributeName: mark the name of the attribute to be animated;
from: mark the start point;
to: mark the end point;
dur: marks how long the animation will take to be completed;
fill: marks if the "to" value will be "remove" or "freeze", if it will be reset or will be hold;
repeatCount: marks how many times the animation will happen or it will be "indefinite";

The "path, /path" tag defines a path, using the "d" attribute we can add some information:
M: moveto;
L: lineto;
H: horizontal lineto;
V: vertical lineto;
C: curveto;
S: smooth curveto;
Q: quadratic Bézier curve;
T: smooth quadratic Bézier curveto;
A: elliptical Arc;
Z: closepath;
M is the start point and Z the end point;
