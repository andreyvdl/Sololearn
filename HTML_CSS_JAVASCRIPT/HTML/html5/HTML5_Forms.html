Some things have changed with HTML5;
Now you can add a "autofocus" attribute to focus on that input after the page load;
Also the "required" attribute can be used to force the user to fill a input;

Theres some new input types:
color;
date;
datetime;
datetime-local;
email;
month;
number;
range;
search;
tel;
time;
url;
week;

Theres also some new attributes:
form;
formaction;
formenctype;
formmethod;
formnovalidate;
formtarget;
height and width;
list;
min and max;
multiple;
pattern (regexp);
placeholder;
step;

The "datalist, /datalist" creates a datalist container;
The "option, /option" tag creates a option that will be on the datalist;
The input box will load the datalist and the options inside of the datalist as a drop-down list;
